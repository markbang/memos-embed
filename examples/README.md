# Blog integration examples

These examples show how to embed Memos posts in common publishing stacks.

- `next-mdx/`: Next.js App Router with server-fetched memo data passed into the React wrapper
- `mdx-components/`: reusable MDX component pattern for React-based blogs
- `astro-blog/`: Astro + MDX usage with a lightweight wrapper component
- `static-html/`: iframe and Web Component snippets for static HTML or CMS pages

They are intentionally minimal reference implementations, not full runnable starters.
