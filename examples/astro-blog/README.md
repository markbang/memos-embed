# Astro blog example

Use this pattern when your site is built with Astro and you want to embed memo cards inside pages or MDX content.

## What it demonstrates
- fetch memo data in Astro frontmatter
- render the core HTML snippet directly for SSR-friendly output
- theme the embed with CSS variables from your Astro site

This approach works well when you want static output and do not need a React runtime on the page.
